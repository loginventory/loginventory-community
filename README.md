# LOGINventory Community

A community library of dashboards, queries, and documents created in [LOGINventory](https://www.loginventory.de/).
Browse, download, and reuse content shared by other LOGINventory users.

---

## 📖 About

This repository is a curated collection of ready-to-use LOGINventory content contributed by users and the LOGINventory team. Instead of building everything from scratch, you can pick what fits your needs and import it directly into your own environment.

## 📂 What you'll find here

- **[Dashboards](./dashboards)** — Visual overviews for asset management, license tracking, hardware lifecycle, security insights, and more
- **[Queries](./queries)** — Pre-built queries to answer common questions about your IT inventory
- **[Documents](./documents)** — Report templates and document layouts ready to be adapted

Each contribution lives in its own folder and includes a short description, a preview (where applicable), and instructions on how to use it.

## 🚀 How to use

1. Browse the folders or use GitHub's search to find content that fits your needs.
2. Download the file(s) for the dashboard, query, or document you want to use.
3. Import them into your LOGINventory installation.
4. Adjust them to your environment as needed.

> 💡 Detailed import instructions can be found in the LOGINventory manual ([English](https://www.loginventory.info/documentation/9/en/) | [Deutsch](https://www.loginventory.info/documentation/9/de/)).

## 🤝 Contributing

Have you built something useful in LOGINventory that others might benefit from? We'd love to include it here!

To contribute:

1. Fork this repository
2. Add your content in the matching folder (`dashboards`, `queries`, or `documents`)
3. Include a short `README.md` describing what your content does and how to use it
4. Open a pull request

Please make sure your contribution does **not** contain any sensitive or customer-specific data (e.g. real device names, IP addresses, user accounts, license keys).
